# LightGBM Prediction Model

This repository contains Python code in Jupyter Notebook format for the LightGBM prediction model used in the rice arsenic and cadmium analysis.

## Files

- `LightGBM_Prediction_Model.ipynb`: Python code in Jupyter Notebook format for model training, prediction and evaluation.
- `example_data.csv`: simulated example dataset used to demonstrate how to run the code.
- `requirements.txt`: list of required Python packages.

## System requirements

The notebook was tested using Python 3.10 on Windows 11.

Required Python packages include:

- pandas
- numpy
- scikit-learn
- lightgbm
- matplotlib
- seaborn
- scipy
- openpyxl
- jupyter

No non-standard hardware is required. The notebook can be run on a normal desktop or laptop computer.

## Installation

Install the required Python packages using:

```bash
pip install -r requirements.txt
```

Typical installation time is approximately 5–10 minutes on a normal desktop computer with internet access.

## Example data

A simulated example dataset, `example_data.csv`, is provided to demonstrate how to run the LightGBM prediction model.

The simulated dataset has the same general data structure as the input data used in the analysis, but it does not contain the original research data.

The original input data used in the manuscript were compiled from published literature and publicly available datasets, as described in the Methods and Supplementary Tables of the manuscript.

## How to run the example

Open the Jupyter Notebook:

```bash
jupyter notebook LightGBM_Prediction_Model.ipynb
```

Then run the cells sequentially.

The notebook reads the example dataset `example_data.csv` and performs model training, prediction and evaluation using the LightGBM model.

## Expected output

The expected output includes:

- model training results;
- prediction results;
- model performance metrics;
- feature importance results or plots generated by the LightGBM model.

The expected run time for the example is approximately 5–20 minutes on a normal desktop or laptop computer, depending on the computer configuration.

## How to run the code on user-provided data

To run the notebook on user-provided data, replace `example_data.csv` with the user's own input data file in the data-loading cell of the notebook.

The user-provided data should contain the same variable names and data structure as the example dataset.

## Reproduction

Running all cells in the notebook reproduces the LightGBM modelling workflow used in the manuscript.

Because the included `example_data.csv` file is a simulated dataset, the numerical results generated from the example data are for demonstration purposes only and are not expected to exactly match the results reported in the manuscript.
