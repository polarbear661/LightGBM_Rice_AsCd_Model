# LightGBM Prediction Model

This repository contains the Jupyter Notebook code used for the LightGBM prediction model in our manuscript submitted to Nature Food.

## File

- `LightGBM_Prediction_Model.ipynb`: Jupyter Notebook for model training, prediction and evaluation.

## Software

The code was written in Python and run in Jupyter Notebook.

Required Python packages include:

- pandas
- numpy
- scikit-learn
- lightgbm
- matplotlib
- seaborn
- scipy
- openpyxl

## Data

The input data used in the analysis were compiled from published literature and publicly available datasets, as described in the Methods and Supplementary Tables of the manuscript.

Install the required packages and open the notebook:

```bash
jupyter notebook LightGBM_Prediction_Model.ipynb
